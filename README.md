# Qt-Zhku-Client

算法学累了，不如来写qt?

[![standard-readme compliant](https://img.shields.io/badge/readme%20style-standard-brightgreen.svg?style=flat-square)](.)

***

本仓库包含以下内容：

1. 重造的一个仲恺教务客户端，原本用python写的太烂了
2. 基于QT+爬虫形式，效率较低，没有官方API
3. 语法垃圾,实现的很笨,轻喷,喷就马上改

开发文章会在公众号更新
